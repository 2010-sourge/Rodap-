# 🦶 Responsive Footer — HTML & CSS

Um componente de **rodapé (footer) responsivo** moderno com tema escuro, construído com **HTML e CSS puro**. Inclui seções de informações, serviços, formulário de newsletter e redes sociais.

---

## ✨ Preview

O footer é dividido em **três colunas**:

| Coluna 1 | Coluna 2 | Coluna 3 |
|---|---|---|
| About us + Contact us | Our Services | Subscribe us |

---

## 🛠️ Tecnologias Utilizadas

- **HTML5** — estrutura semântica com seções bem definidas
- **CSS3** — dark theme, Grid/Flexbox, hover effects e estilização de formulário

---

## 📁 Estrutura do Projeto

```
footer-design/
├── index.html
└── style.css
```

---

## 🚀 Como Executar

1. Clone ou baixe este repositório
2. Abra o arquivo `index.html` no seu navegador

Nenhuma instalação ou dependência necessária!

## 📋 Seções do Footer

### 1. About Us
Descrição do projeto ou empresa, seguida das informações de contato com telefone e e-mail.

### 2. Our Services
Lista de links de serviços oferecidos:
- Mobile Application Design
- Wire framing & Prototyping
- Web Design, Development
- Web User Interface Design
- Web UX Design, Research
- Theme Development, Design

### 3. Subscribe Us
Formulário de newsletter com campo de e-mail e botão **Send** em laranja, além dos ícones de redes sociais (Facebook, Twitter, Instagram, YouTube, LinkedIn).

---

## ⚙️ Personalização

| Elemento | Como alterar |
|---|---|
| Cor de destaque (laranja) | Altere `#e8442a` no CSS |
| Cor do fundo | Altere o `background` do `footer` |
| Links de serviços | Edite os `<li>` na seção `.services` |
| Informações de contato | Edite telefone e e-mail no HTML |
| Redes sociais | Adicione ou remova ícones em `.social-icons` |

---

## 📌 Funcionalidades

- [x] Layout em 3 colunas com CSS Grid
- [x] Tema escuro (`#1a1a1a` / `#222`)
- [x] Cor de destaque laranja/vermelho nos títulos e botão
- [x] Formulário de newsletter estilizado
- [x] Ícones de redes sociais com hover effect
- [x] Barra de copyright no rodapé
- [x] 100% HTML e CSS — sem JavaScript
